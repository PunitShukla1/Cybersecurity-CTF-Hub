Prepare for an elite challenge! Follow the instructions to set up your environment.

Setup Instructions

  1. 📥 Download the lab tar file.
  2. 📦 Extract the tar file using the command: tar -xvf [file_name]
  3. 📂 Navigate to the labs/peh_web_lab directory.
  4. 🐳 Install Docker: sudo apt install docker.io
  5. 🛠️ Install Docker Compose: sudo apt install docker-compose
  6. 📁 Go to the directory containing docker-compose.yml
  7. 🚀 Start the lab: sudo docker-compose up
  8. 💡 Optional: Run in the background with sudo docker-compose up -d
  9. 🌐 Access the challenge: Start Challenge
